CAPABILITIES

⌨
DOS Terminal UI
Apple II inspired CRT interface. Phosphor glow, bezel frame, drive status display. Every command runs through the BOB reasoning layer.
⬡
BOB Integration
BOB Orchestrator handles git intelligence — diff summaries, commit recommendations, branch strategy, merge analysis.
⛓
WORM Sealing
Every command and response is appended to a local WORM chain. SHA-256 sealed, append-only. Your git history has a history.
🛡
Catcode Guardrail
Sovereign content policy enforced at the prompt layer. Checks for secret leakage, unsafe force-push patterns, and out-of-bounds operations.
◎
Ollama + OpenAI
Configurable endpoint. Default: Ollama localhost. Swap to any OpenAI-compatible API — LM Studio, Groq, Together, OpenRouter.
∴
Arena Mode
Run git crawlers across multiple repos simultaneously. Results logged to WORM chain. BOB ranks by commit health, catcode score, and diff quality.
